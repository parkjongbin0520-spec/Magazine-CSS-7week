## 과제 목표

2차원 레이아웃의 끝판왕인 **CSS Grid**와 웹 페이지에 생동감을 불어넣는 **Transition & Animation**, 그리고 스마트한 **CSS 변수(var)** 활용법에 대해 학습하고 지금까지 배운 내용을 총동원하여 나만의 멋진 매거진 커버를 완성하기.

## 필수 사항

1. Grid 레이아웃 적용
  
  `grid-template-columns` 를 사용하여 전체 레이아웃과 내부 기사 카드 영역이 화면 캡처(미리보기) 처럼 다단으로 잘 나뉘도록 배치해주세요.
  
  모바일 화면(폭이 좁아질 때)에서는 1단으로 변하도록 Media Query도 확인합니다.
  
2. Animation 효과 적용
  
  `@keyframes` 를 이용해 화면 상단의 텍스트가 흘러가는 효과(ticker)와 기사 카드의 뱃지(NEW/HOT)가 깜빡이거나 둥둥 뜨는 효과가 제대로 작동해야합니다.
  
3. 제출 방법 및 유의 사항
  
  마크다운(Markdown) 포맷을 사용하여 작성 후 PDF로 변환하여 제출합니다
  
  **GitHub Pages 배포 링크** (저장소가 아닌 Index 페이지로 이동)
  
  본인이 추가/수정한 **CSS 코드 블록** 을 ````` 를 활용하여 코드 포함
  
  Grid 레이아웃이 다단으로 적용된 **PC 및 모바일 전체 화면 캡처**
  
4. AI 활용 지침
  
  본 과제 수행 시 Gemini, ChatGPT 등의 생성형 AI 활용이 가능합니다. 단, 디자인 아이디어나 코드 작성에 AI의 도움을 받은 경우 문서 하단에 어느 부분에 어떻게 활용했는지 명시하고, AI와 나눈 대화 내역 전체를 캡처하여 PDF에 함께 첨부해야 합니다. *(출처 미기재 및 단순 복사 제출은 미제출 처리될 수 있습니다.)*
